# Mechanistic Interpretability — GPT-2
Ongoing experiments probing the internal circuits of GPT-2 small using Transformer Lens

## Experiments for Learning! 
| Caching Activations with Visualisations
| Ablation : It's effect on loss 
| Activation Patching on Indirect Object Identification

## General (non-technical) Learnings 
- If you love being a detective, this is for you.
- The observations with any chosen prompt can be significant, but I think it is more important to spend time on the prompt you are choosing to guage the circuits on
- The techniques to probe generally exist, but it is important to form the right research question
- This technique is a combination of inferences and description of the model, we can treat most of our observations for both
